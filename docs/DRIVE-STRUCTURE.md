# Estrutura Google Drive

> Guia para criar e manter a pasta no Google Drive
> Atualizado: 2026-04-17

## Pasta Raiz

```
MSC Control Center/
```

## Subpastas

### Projetos/
Documentos por projeto para compartilhar com clientes ou equipe.
Criar uma subpasta por projeto conforme necessidade.

```
Projetos/
  arquimedes/       # Docs do LMS
  claira/           # Docs do sistema legal
  detran-contratos/ # Docs DETRAN
  msc-crm/          # Docs do CRM
```

### Clientes/
Documentos de clientes — propostas, contratos, briefings.

```
Clientes/
  cliente-nome/
    proposta.pdf
    contrato.pdf
```

### Templates/
Templates reutilizaveis para documentos recorrentes.

```
Templates/
  daily-log-template.md
  weekly-review-template.md
  project-brief-template.md
  proposta-comercial-template.docx
```

### Documentacao/
Exports dos DOC-*.md do vault Obsidian em formato PDF ou MD.

```
Documentacao/
  DOC-projetos.pdf
  DOC-ferramentas.pdf
  DOC-obsidian.pdf
```

### Rotinas/
Checklists e rotinas compartilhaveis.

```
Rotinas/
  checklist-diario.md
  checklist-semanal.md
  onboarding-novo-projeto.md
```

## Como Manter

1. **Docs de projeto**: criar subpasta em `Projetos/` quando precisar compartilhar algo
2. **Templates**: adicionar novos templates conforme surgir necessidade
3. **Documentacao**: re-exportar DOC-*.md quando houver mudancas significativas
4. **Rotinas**: manter checklists atualizados
